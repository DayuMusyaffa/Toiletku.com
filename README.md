# 🚽 ToiletKu

![ToiletKu Banner](https://img.shields.io/badge/ToiletKu-Dashboard%20Ramah-blue?style=for-the-badge&logo=react)

Selamat datang di **ToiletKu**, dashboard toilet pintar yang dirancang untuk semua usia dan kebutuhan. Aplikasi ini menampilkan status fasilitas, monitoring ruang, panduan akses, dan pengalaman pengguna yang nyaman — semuanya dalam balutan desain modern React + TailwindCSS.

> **ToiletKu** adalah solusi UI untuk memantau dan mengelola fasilitas toilet dengan cepat, jelas, dan ramah.

---

## ✨ Fitur Utama

- 🚻 **Monitoring toilet real-time**: lihat status setiap toilet dengan indikator bersih, penuh, atau tersedia.
- 📊 **Dashboard visual**: ringkas informasi penting dengan grafik dan statistik yang mudah dibaca.
- 🔒 **Login pengguna**: akses halaman khusus dengan autentikasi sederhana.
- 🧭 **Panduan ramah pengguna**: tampilkan informasi fasilitas dan petunjuk penggunaan yang jelas.
- 🌈 **Desain inklusif**: cocok untuk semua usia dan kebutuhan akses publik.

## 🚀 Instalasi & Jalankan

1. Install dependensi:

```bash
npm install
```

2. Jalankan server development:

```bash
npm run dev
```

3. Buka hasilnya di browser pada `http://localhost:5173`

4. Build produksi:

```bash
npm run build
```

5. Preview hasil build:

```bash
npm run preview
```

---

## 🧩 Struktur Teknologi

- **React** untuk UI interaktif
- **TailwindCSS** untuk styling cepat dan responsif
- **PostCSS** untuk optimasi stylesheet
- **Vite** untuk pengalaman development yang ringan
